
# Eat-N-Split

# Features :
* Split bill with your friends.
* Add new friends.

# Used technologies
* React.js
* Css


## Run Locally

Clone the project

```bash
  git clone https://github.com/Ankit73-bit/Eat-N-Split.git
```

Install dependencies

```bash
  npm install
```

Start the server

```Terminal (backend)
  npm run start
```
